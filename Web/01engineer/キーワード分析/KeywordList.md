# KW選定リスト（01engineer）

KW選定〜記事執筆の進め方は `Strategy.md` 参照。

## ステータスの見方
- 未検討: KW候補として挙げただけの状態
- 選定済み: 記事化することを決定した状態
- 執筆中: `Articles/` 配下で執筆を進めている状態
- 公開済み: 記事を公開し、URLを記録した状態

## 検索意図の考え方（カテゴライズ方針）
- 大きい目的: なぜこのテーマで検索したか（転職検討・スキルアップ・業界研究など）
- 小さい目的: そのページで具体的に何を知りたいか
- 層: 顕在層／準顕在層／潜在層／認知層／未認知層（下記フレームワーク参照）

| 層 | 状態 | KW例 |
|---|---|---|
| 顕在層 | 今すぐ応募・比較したい | 「PRUM 評判」「未経験 エンジニア 求人」「SES 転職 おすすめ」 |
| 準顕在層 | 転職を考え始めている | 「未経験 エンジニア 転職」「SES やめたほうがいい」「IT業界 転職」 |
| 潜在層 | 悩みはあるが転職は決めていない | 「新人 エンジニア 辛い」「プログラミング 向いてない」「仕事 覚えられない」 |
| 認知層 | 興味・情報収集段階 | 「ChatGPT 使い方」「AI 勉強法」「プログラミング 独学」 |
| 未認知層 | 課題を自覚していない | SNS流入中心（検索KWでは基本捕捉できない） |

「やめとけ」「オワコン」「末路」「闇」等のネガティブワードは、額面通り「否定的」と読まず、指名検索（職種名）とセットになっている場合は**その職種がすでに転職の選択肢に入っている証拠**として扱い、準顕在層〜顕在層に寄せて判定した。
